# Cognifyz Frontend Internship

This repository showcases the tasks and projects completed as part of the **Front-End Development Internship at Cognifyz Technologies**.  
The project is structured into multiple levels, each focusing on different core frontend concepts and hands-on implementation.

## Project Structure

The repository is organized in a level-wise manner for clarity and easy navigation.

###  Level 1 – HTML & CSS
- Basic portfolio layout
- Structured HTML content
- Inline and external CSS styling

###  Level 2 – Responsive Design & JavaScript
- Responsive card layouts using CSS
- JavaScript interactivity
- Background color change using JavaScript

###  Level 3 – API Integration & Form Validation
- Fetching user data from a public API
- Dynamically displaying data on the UI
- Contact form with validation

### Level 4 – Bootstrap & Sass
- Bootstrap framework implementation
- Responsive grid system and cards
- Sass variables and nesting

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Sass
